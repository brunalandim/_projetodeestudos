Introdução Jquery O que é?

Jquery é uma biblioteca (não é um Framework), que contém uma série de blocos em Javascript e permite, que simplifica a programação, tanto em seleção de elementos como em invocação de funções, alcance de objetos, uso de métodos e propriedades.

A forma como se usa Javascript puro, escrito em sua sintaxe padrão, é conhecida como Vanilla Javascript.

Diferente dessa forma, mas utilizando-se dela para compor objetos, funções e métodos, o JQuery traz uma sintaxe específica e muito simplificada.

A inserção do Jquery no documento HTML será diferente do costume atual adotado para <script> em Vanilla, ou seja, o <script> para vincular o arquivo do Jquery será colocado na seção <head>.

Para começar, é necessário trazer - preferencialmente - a última versão comprimida do Jquery e salvá-la em uma pasta ou na raiz.

Importante: pode acontecer de o navegador abrir o código diretamente na tela. Selecione, copie, crie um arquivo JS manualmente e cole o código. Salve ao final.
