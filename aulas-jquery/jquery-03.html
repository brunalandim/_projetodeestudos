Introdução Jquery O que é?

Certamente, neste ponto podemos lembrar do innerHTML. Ele surge aqui de forma distinta, porém muito intuitiva.

$( "#btMudaTeste" ).html();

Da mesma forma que utliza-se o "innerHTML" no Javascript Vanilla, é possivel atribuir valores nessa propriedade usando essa sintaxe acima, através do JQuery. lembre de que é necessário estar vinculado ao arquivo do JQuery na tag.

<script src="js/jquery-3.4.1.min.js"></script>

Observe os elementos abaixo. Ao clicar no botão "Mudar" o texto dos 3 elementos serão alterados.
